# Installation

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
playwright install
cd rust_bindings
maturin build --release
pip install target/wheels/*.whl

cd plugins
git clone https://github.com/krystianbajno/techka-secret
```
